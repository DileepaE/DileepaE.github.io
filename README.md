## üöÄ How to Create Your Own
If you want to develop a similar website, follow these steps:

1. **Fork this repository:** Click the "Fork" button at the top right of this page to create a copy in your own GitHub account.
2. **Clone to RStudio:** Copy your new repository URL and start a "New Project -> Version Control -> Git" in RStudio.
3. **Customize your Info:** - Open `index.qmd` to update your name, bio, and image.
   - Edit `_quarto.yml` to change the website title and navigation links.
4. **Render:** Click the **Render** button in RStudio to generate the website files.
5. **Deploy:** Push your changes back to GitHub and enable "GitHub Pages" in your repository settings (pointing to the `docs` folder or the `gh-pages` branch).
